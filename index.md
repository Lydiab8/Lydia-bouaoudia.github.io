## 🎯 Objectif
Concevoir un agent intelligent capable d’optimiser la gestion énergétique d’un système industriel complexe.

---

## ⚙️ Travaux réalisés

- Développement d’un agent d’optimisation basé sur le **Reinforcement Learning**
- Simulation d’un environnement industriel (machines + consommation énergétique)
- Intégration de modèles prédictifs (LSTM) pour anticiper la demande énergétique
- Conception d’un **Energy Management System (EMS) intelligent**
- Prise en compte de contraintes réelles (production, stockage, coût énergie)
- Développement d’une interface interactive avec **Streamlit**
- Collaboration avec équipes industrielles et data engineers

- Gymnasium (environnement simulé)
- LSTM pour séries temporelles
- Optimisation sous contraintes
- Pipeline de données industrielles

---

## 📊 Résultats

- Meilleure allocation simulée de l’énergie
- Réduction des pics de consommation
- Amélioration de l’utilisation des énergies renouvelables
